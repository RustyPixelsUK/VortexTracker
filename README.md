# Vortex Tracker 3.0

The best music tracker for 8bit AY/YM chips.

## New in Version 3.0
- C# Port from Delphi
- OpenGL rendering
- OpenAL audio
- Spectrum and VU bars
- 3AY support
- Built-in OrGen Editor
- YM2149 Synth slave support
- Spectrum Next slave support
- Plugins
